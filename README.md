# Bookworm
A simple Laravel project for books.

## Dependencies
- PHP 8.1
- Laravel 9
- Composer 2
- Node 19
- Tailwind
- DaisyUI
- PostCSS

## Setup

```bash
$ composer install
```

```bash
 $ npm install
```

```bash
$ npm run dev
```

In root directory add a ``.env`` file with your configuration

Serve with artisan

```bash
$ php artisan serve
```
